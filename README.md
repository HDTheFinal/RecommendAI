# RecommendAI
##  1. 크롤링
### 1.1. 크롤링 환경 변수 생성
1. `venv` 폴더를 생성한다.
2. 가상 환경을 생성한다.
```
// 가상환경 생성
python -m venv crawling

// 가상환경 활성화
call crawling/Scripts/activate

// 가상환경 종료
deactivate
```
3. 삭제할 시에는 폴더를 삭제한다.

### 1.2. 크롤링 모듈 설치
```
// selenium
pip install selenium

// BeautifulSoup
pip install bs4
```
